In the app, you might encounter a specific code for which package the customer wants to buy. Here is the definition of the code

NAMING LEGEND
- XT_XCUP
(X = any number, T = Tiered Cake, CUP = Cupcakes)

Example:
- 3T_12CUP
- This means the customer wants the 3 Tiered Cake with 12 cupcakes package
